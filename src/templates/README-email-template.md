# Certificate Email Template

## Overview
Professional, responsive email template for digital certificate delivery system.

## Subject Line
```
Your Certificate for {{course_name}} is Ready
```

## Dynamic Variables

The following placeholders are supported in the template:

- `{{student_name}}` - Recipient's name
- `{{course_name}}` - Name of the course/certification
- `{{certificate_id}}` - Unique certificate identifier
- `{{issue_date}}` - Date when certificate was issued
- `{{certificate_link}}` - Direct download link for the certificate
- `{{organization_name}}` - Name of the issuing organization

## Features

- ✅ Mobile-responsive design
- ✅ Compatible with Gmail, Outlook, Yahoo
- ✅ Inline CSS (no external stylesheets)
- ✅ Accessible design (readable fonts, proper contrast)
- ✅ Professional enterprise SaaS styling
- ✅ Clear call-to-action button
- ✅ Certificate details card
- ✅ Footer with organization info and disclaimer

## Usage

1. Replace all placeholder variables with actual values
2. Replace support email in footer: `support@example.com`
3. Update logo placeholder with actual organization logo
4. Test across different email clients before sending

## Customization

### Logo
Replace the logo placeholder in the header section with an actual image:
```html
<img src="LOGO_URL" alt="Organization Logo" style="max-width: 120px; height: auto;">
```

### Colors
Main colors used:
- Primary: #0ea5e9 (cyan-500)
- Secondary: #06b6d4 (teal-500)
- Text: #1f2937 (gray-900)
- Background: #ffffff, #f9fafb

### Support Email
Update the support email link in the footer section.
