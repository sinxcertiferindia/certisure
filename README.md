# Certiflow Pro

## Project Overview
Certiflow Pro is a web-based certificate management platform built using modern frontend technologies.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Local Setup

### Prerequisites
- Node.js
- npm

### Steps
```sh
git clone <REPOSITORY_URL>
cd certiflow-pro
npm install
npm run dev
